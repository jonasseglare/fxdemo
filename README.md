# fxdemo

Illustrates how to get JavaFX working in Clojure, 
especially interactively in the REPL.

## Usage


```
fxdemo.example2> (main-async)
#future[{:status :pending, :val nil} 0x7d800d2d]
fxdemo.example2> 
fxdemo.core> (testrun-gui)
nil
fxdemo.core> (testrun-gui)
nil
fxdemo.core> 
```
